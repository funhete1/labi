Documento HTML

Documentos HTML possume uma estrutura:
<"html"> --> indica o inicio do seu documento
<"head"> --> indica a parte inicial do seu documento, como por exemplo titulo, resumo, merdas do genero.
<"body"> como o nome sugere e a parte do documento em que o codigo e escrito, ou seja acaba por ser msm o corpo do documento.

Elementos HTML

todas as marcas sao usadas com "<>" as marcas sao usadas para dar caracteristicas ao documento se que seja necessario o uso de linguagens como CSS e Javascript, dentre as marcas mais basicas temos:
<"p"> para paragrafos
<"br/"> para pular linhas
<"title"> determina o titulo do seu documento
<"b"> poem o texto em negrito
